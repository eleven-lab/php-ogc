## Composer

### GitHub (public)
Add package to package.json:

    "require": {
        "elevenlab/php-ocg": "~1.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/elevenlab/php-ocg"
        }
    ]
    
### BitBucket (private)

    require:{
    "elevenlab/php-ocg":"dev-master"
    }

    "repositories": [
        {
            "type": "vcs",
            "url": "git@bitbucket.org:elevenlab/php-ocg.git"
        }
    ]
